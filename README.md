# 📈 Daily Sales Registry 

 

A lightweight, browser-based application designed to manage daily sales without the need for a backend. It leverages the power of the browser to keep data secure and accessible. 

 

--- 

 

## 🚀 Overview 

 

This tool was created to provide a fast and reliable way to log sales locally. By using the browser's internal storage, users can keep track of their business performance with zero setup and no server costs. 

 

## 🛠️ Tech Stack 

 

- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid) 

- **Logic:** JavaScript (Vanilla ES6+) 

- **Storage:** LocalStorage API (Client-side persistence) 

 

--- 

 

## ✨ Key Features 

 

- **Local Data Persistence:** Your information is saved directly in your browser using `localStorage`, so it stays there even if you refresh or close the page. 

- **No-Server Architecture:** Fast, private, and works entirely on the client side. 

- **Dynamic List Management:** Add and track sales entries in real-time with an intuitive UI. 

- **Lightweight & Portable:** Can be hosted on GitHub Pages with a single click. 

 

--- 

 

## ⚙️ How it works 

 

The application uses the `localStorage` web API to store data as a JSON string. This allows for: 

1. **Instant saving** upon entry. 

2. **Persistent state** between sessions. 

3. **Privacy**, as the data never leaves the user's computer.
